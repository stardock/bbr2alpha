# bbr2alpha
TCP BBR2 / bbr v2alpha  


https://github.com/google/bbr/tree/v2alpha  
2021-01-15更新  
Linux内核版本5.10.0  
自编译，适用于Ubuntu/Debian  

默认配置  
net.core.default_qdisc = fq_pie  
net.ipv4.tcp_congestion_control = bbr2  

安装后重启即可，无需其他操作
